Les films les plus populaires

Ce site web regroupe les données scrappées des films les plus populaires d'après le classement de IMDb. Vous pouvez retrouver le classement sur ce lien.

Vous pouvez chercher un film par nom d'acteur, ou d'après le nom d'un personnage ou encore via le nom d'un réalisateur mais également en mettant son titre. Vous obtiendrez alors toutes les informations principales concernant les films recherchés. La section Catégorie permet également d'afficher les meilleurs films par catégorie. Ces recherches vous permettront de savoir si un acteur particulier a joué dans un film à succès ou encore de savoir si un de vos films préférés fait partie des tendances.

Par exemple :
- Nom d'acteur : Chris Hemsworth
- Titre de film : Soul

IMDb est une base de données américaine, c'est pourquoi toutes les recherches doivent être en anglais et les résultats le seront également.
